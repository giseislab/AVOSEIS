Installation procedure:

First installation only:
1 cd adm/coldstart
2 make
3 cd ../..
4 make firsttime

All other times (Updates)
1. source $AVOSEIS/setup.sh or $AVOSEIS/setup.csh to set AVOSEISMAKE
2. chmod -R a+w $AVOSEIS
3. cd lib/makerules
4. make Include
5. cd ../..
6. make Include
7. make install
8. make scripts
9. make perlmods
10. make schemas
11. make pf_localmods
12. make host # real-time servers only.
### 13-15 REAL_TIME SERVERS ONLY
13. chmod -R a+w $INTERNALWEBPRODUCTS
14. make html
15. chmod -R a-w $INTERNALWEBPRODUCTS
