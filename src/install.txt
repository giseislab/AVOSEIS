Installation procedure:

First installation only:
1 cd adm/coldstart
2 make
3 cd ../..
4 make firsttime

All other times ("Updates")
* source $AVOSEIS/setup.sh or $AVOSEIS/setup.csh to set AVOSEISMAKE variable *
1 cd lib/makerules
2 make Include
3 cd ../..
4 make Include
5 make install
6 make scripts
7 make perlmods
8 make schemas
9 make pf_localmods

Real-time servers only
10 make host
11 make html
